📝 INITIAL SETUP (Do Once)
═══════════════════════════════════════════════════════════════════════

1. Get Test PyPI Token
   → https://test.pypi.org/manage/account/token/
   → Name: seer-github-actions
   → Copy token (starts with pypi-)

2. Get Production PyPI Token
   → https://pypi.org/manage/account/token/
   → Name: seer-github-actions
   → Copy token (starts with pypi-)

3. Add to GitHub
   → https://github.com/ryanbieber/seer/settings/secrets/actions
   → Add: TEST_PYPI_TOKEN = (test token)
   → Add: PYPI_TOKEN = (production token)

═══════════════════════════════════════════════════════════════════════
🚀 RELEASE PROCESS
═══════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│ Step 1: Test Locally                                               │
└─────────────────────────────────────────────────────────────────────┘
./test_deployment.sh

┌─────────────────────────────────────────────────────────────────────┐
│ Step 2: Update Version                                             │
└─────────────────────────────────────────────────────────────────────┘
Edit these files:
  • pyproject.toml → version = "0.1.0"
  • Cargo.toml → version = "0.1.0"

┌─────────────────────────────────────────────────────────────────────┐
│ Step 3: Commit and Tag                                             │
└─────────────────────────────────────────────────────────────────────┘
git add pyproject.toml Cargo.toml
git commit -m "Release v0.1.0"
git tag v0.1.0
git push origin master --tags

┌─────────────────────────────────────────────────────────────────────┐
│ Step 4: Create GitHub Release                                      │
└─────────────────────────────────────────────────────────────────────┘
https://github.com/ryanbieber/seer/releases/new
  • Choose tag: v0.1.0
  • Title: v0.1.0
  • Add release notes
  • Click "Publish release"

┌─────────────────────────────────────────────────────────────────────┐
│ Step 5: Watch Automated Deployment                                 │
└─────────────────────────────────────────────────────────────────────┘
https://github.com/ryanbieber/seer/actions
  ✅ Tests run
  ✅ Wheels built
  ✅ Upload to Test PyPI
  ✅ Test installation
  ✅ Upload to Production PyPI
